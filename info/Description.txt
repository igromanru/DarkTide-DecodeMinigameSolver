[font=Verdana][size=4][b]Description[/b][/size][/font]
The mod basically plays the Decode Symbols minigame for you.
Warning! It might be inaccurate sometimes.
Current set up works for me like 99% of the time. If you have any issues, check out the “Settings” section below and report your experience in the comments.

[font=Verdana][size=4][b]Requirements[/b][/size][/font]
[size=3]- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url]
- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url][/size]

[font=Verdana][size=4][b]Installation[/b][/size][/font]
1. Read descriptions and install [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url] and [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url]
2. Download and install latest version of the mod with Vortex from the [b]Files[/b] tab.
3. Make sure the mod is enabled in Vortex's [b]Load Order[/b] tab.
4. In game you can disable options in the [b]Mod Options[/b] menu.

[font=Verdana][size=4][b]Settings[/b][/size][/font]
The mod has two parameter that can be changed. [b]Interaction Cooldown[/b] and [b]Target Precision[/b].
[font=Verdana][b]Interaction Cooldown[/b][/font]
[size=2]Default value: 150[/size]
It's the wait time in milliseconds between each interaction click.
The cooldown is needed to prevent double clicks.
[font=Verdana][b]Target Precision[/b][/font]
[size=2]Default value: 4[/size]
The value determines how close the minigame cursor (the moving box) has to be to the target symbol's center before the mod interacts with it.
The higher the value, the closer the cursor has to be to the center before the mod triggers an interaction.
